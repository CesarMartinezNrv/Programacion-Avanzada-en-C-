A2. (5 pts) - 4
¿Cuándo usarías paso por VALOR y cuándo paso por REFERENCIA?
Da un ejemplo concreto para cada caso que justifique tu elección.

Respuesta:
Se utilizaria paso por valor cuando queremos una nueva variable o un return que nos genere un nuevo dato.
Paso por referencia se utilizaria en caso de que se quiera modificar una variable x directamente sin generar una variable local en una funcion.
Ejemolo:
Si se quiere modificar un arreglo el paso por valor conllevaria un mayor uso de recursos porque se deberia crear una copia del arreglo, 
mientras que el paso por referencia modifica el arreglo mismo y no se necesita crear una copia.

A3. (5 pts) - 5
¿Qué es una función recursiva?
Nombra sus DOS partes obligatorias y explica qué ocurre
si falta alguna de ellas.

Respuesta:
Una funcion recursiva es una funcion que se llama asi si misma.
1. Debe existir un caso base para que cuando se llame a si misma en algun punto pare ese llamado y no exista un bucle infinito.
2. Cada que se ingresa a la funcion nuevamente el parametron debe cambiar.

Si falta alguna de estas el bucle sera infinito ya que nunca caeria en el caso base y nunca se terminaria el bucle de llamar a si misma.
